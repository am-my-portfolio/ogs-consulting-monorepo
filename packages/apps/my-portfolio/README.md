# Vue3 App with Vite and TailwindCss

This is a Vue3 Single Page Application deployed using Github Pages

## Running Locally

1. Install the dependencies with npm:

```bash
pnpm install
```

2. Run the app:

```bash
pnpm start
```

## Deployed

- do this once
  `pnpm run build`
  `git subtree push --prefix dist origin gh-pages`

- after every change
  `pnpm run deploy`
